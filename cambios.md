# v2.0

## novedades

### adiciones

- ajuste de opción: similar a enumsetting, pero abre un marco de selección, igual que listsetting.

#### módulos nuevos:

- resaltado de bloques: resalta bloques de cierto tipo renderizando una caja con su forma, visible a través de paredes
- delay de paquetes: simula un ping alto añadiendo delay a los paquetes
- inventario plus: te permite mover items a cualquier slot del inventario mediante teclas (ej.: d + tecla para slot 4);
- progreso de minado: muestra el progreso de minado de un bloque de una manera más visual, mediante cajas de color
- mano secundaria: equipa de manera automática un ítem en tu mano secundaria
- moverse en interfaz: te permite utilizar las teclas de movimiento con ciertas interfaces abiertas (configurable)

### arreglos / mejoras

- ahora se pueden mostrar también las coordenadas del nether en el elemento del hud para coordenadas
- ahora ChatPlus utilizará los corchetes [] para mostrar la hora (antes era <>), además de también detectar cuando un jugador se une a la hora de dibujar las cabezas
- ahora puedes elegir el sonido en aviso de baja dur., junto a la altura y volumen de este
- ya no se pueden abrir interfaces del mod estando en pantallas donde se suele escribir (chat, carteles, etc.)
- arreglado algunos problemas con las configuraciones
